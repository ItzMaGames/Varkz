# ⚡ VARKZ

VARKZ é uma **linguagem de programação experimental rodando no navegador**, com sua própria IDE, interpretador em JavaScript e sistema modular de comandos.

O objetivo do projeto é explorar como criar uma linguagem simples, extensível e divertida de usar diretamente no browser.

---

## 🚀 Demonstração

```varkz
var x=10
show x

repeat 3
show "Olá VARKZ"
end
